# LukeWiley-web-project
ICT280 – Project 2
HTML‑Only Website (No CSS)
This project contains a two‑page website created for ICT280 Project 2.
The assignment required building a simple website using HTML only, without any external CSS or JavaScript. All styling is done using inline HTML attributes.

Theme: Study & Organization
The content of this website focuses on my study habits, organization tools, and weekly routines as a student.
Both pages include:

Paragraphs

Lists

Images

Tables

Navigation links

Inline styling
